## Featured Project

### Project94 – Polymorphism and Virtual Functions

This project demonstrates one of the core concepts of Object-Oriented Programming: **Runtime Polymorphism**.

#### Concepts Used

* Base Class
* Derived Classes
* Virtual Functions
* Function Overriding
* Base Class Pointers
* Dynamic Binding

Example hierarchy:

```text
Animal
├── Cat
└── Dog
```

Using a base class pointer (`Animal*`), different derived class implementations are invoked at runtime through virtual functions.
